# TANITA_SILVIFOR

Proyecto organizado por plataforma:

- `PC/`: aplicacion web local y lanzador para Windows.
- `ANDROID/`: proyecto Android, APK debug y scripts de compilacion.
- `IOS/`: proyecto iOS para abrir y compilar en Xcode.

Los datos personales de la bascula quedan ignorados por Git dentro de `PC/TANITA/` y `PC/SilviForTANITA/data/`.
