# iOS SilviForTANITA

Proyecto iOS de SilviForTANITA para abrir en Xcode.

En Windows queda preparado, pero para compilar o subir a TestFlight/App Store hace falta macOS con Xcode.

En un Mac:

```bash
npm install
npm run ios:sync
npm run ios:open
```

La app web fuente esta en `../PC/SilviForTANITA`.
